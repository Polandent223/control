# MiControl PRO v2.0

Versión profesional lista para GitHub Pages y conversión a APK/PWA.

## Incluye
- PIN de acceso.
- Dashboard premium.
- Ingresos y gastos.
- Cuentas múltiples.
- Metas de ahorro.
- Reportes y gráfico mensual.
- Exportar CSV para Excel.
- Reporte PDF/imprimir.
- Respaldo y restauración JSON.
- Modo claro/oscuro.
- Manifest, iconos y service worker.

## Instalación en GitHub
Sube todos los archivos y carpetas al repositorio, reemplazando la versión anterior.
